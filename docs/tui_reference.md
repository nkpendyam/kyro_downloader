# TUI Reference

## Launch
```bash
kyro tui
```

## Screens

### Download Screen
- URL input field
- Fetch Info, Download, MP3 buttons
- Status display and video info

### Queue Screen
- Download queue table with Status, URL, Priority columns
- Start Queue and Clear Completed buttons

## Keyboard Bindings
- `q` - Quit
- `d` - Switch to Download screen
- `k` - Switch to Queue screen
- `r` - Refresh current screen
